    Jira Entegrasyonu

    OAuth 2.0 ile Atlassian bağlantısı kurun, task ve Confluence yönetin.

    Jira Bağlantı Ayarları

    Teknik analizi doğrudan Jira'ya task olarak oluşturun. OAuth 2.0 ile güvenli bağlantı.

    ⚠ "Callback URL is invalid" hatası alıyorsanız Atlassian Developer Console'daki Callback URL, aşağıdaki adresle birebir aynı olmalıdır (tam kopyalayın):
    http://localhost:5002/api/jira/callback
    Kurulum adımları:
    1. developer.atlassian.com/console/myapps → Create → OAuth 2.0 integration
    2. Sol menü → Authorization → OAuth 2.0 (3LO) → Configure → yukarıdaki Callback URL'i yapıştır → Save
    3. Sol menü → Permissions → Jira API + Confluence API ekle
    4. Sol menü → Settings → Client ID ve Secret'ı kopyala → buradaki alanlara gir
    5. "Jira'ya Bağlan" → Atlassian onay ekranı → tamam
